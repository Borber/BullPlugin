BullPlugin

Java plugin loading framework

Quick Start →

Very easy

Just enter the path to load.

Very small

Less than 10KB.

Very lightweight

No other libraries.

Add it in your root build.gradle at the end of repositories

allprojects {
	repositories {
		maven { url 'https://maven.enaium.cn' }
	}
}

Add the dependency

dependencies {
	implementation 'cn.enaium:bullplugin:LATEST'
}
